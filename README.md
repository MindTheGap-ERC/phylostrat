## Requirements and reproduction

Requires Julia, revBayes as `rb`, bash, snakemake, and R. Details for full instructions are given in the `REPRODUCEME.md`.

## Repository structure

* code/
    * bugreports/: bugreports issued for revBayes
    * strat_architectures/: Julia & R code to generate strat architectures, plot them, and do postprocessing
    * bash, revBayes, and R code for phylogenetic simulation and inference, including postprocessing and plotting
* dag/: folder for model dags (content not tracked)
* data/
    * strat/: data from stratigraphic forward models
    * further folders generated by the phylogenetic simulation pipelines, specifically `fbd_base/`, `fbd_strat/`, `fbd_gap_est`, and `fbd_gap_prior`. Contain both simulation data, trace files from inference, and MAP trees
    * convergence_assessment.RData: mcmc convergence statistics
    * post_values.RData: posterior & MAP statistics from traces
* figs/
    * ms/: figures in the main manuscript. Initially empty
    * sm/: figures for supplementary materials. Initially empty
* logs/: folder for log files of mcmc runs (content not tracked)
* .gitignore: untracked files
* LICENSE: Apache 2.0 license text
* README: you are here
* REPRODUCEME.md:  Instructions for reproduction of study
* phylostrat.RProj: R project file for RStudio
* Manifest.toml: Julia deps
* Project.toml: Julia project info
* snakefile_[suffix]: various snakemake workflows, specifically
    * fbd_base: cFBD + CFS simulation and inference
    * fbd_strat: cFBD + MSG and cFBD + FLG simulation and inference
    * fbd_gap_est: sFBDw + FLG simulation and inference
    * fbd_gap_est: sFBDs + FLG simulation and inference
    * tree_summary: generage MAP trees
